A scaffolder for new web projects. One command lays down the file tree, the design tokens, the typography, the deployment config, and a single example page. The defaults are opinionated — Astro or Next, Tailwind, MDX, a single Lora-led prose block, and a hand-picked set of lint rules. The argument is that good defaults are a kind of teaching: you start with what works, and only diverge when you can name the reason.
