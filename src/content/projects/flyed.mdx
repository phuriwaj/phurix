An interface for teachers booking field trips and for venue operators managing capacity. The product was built to work on school-issued Chromebooks and 3G connections, so every page ships a fast static shell, the network calls are minimal, and the heavy interactivity is lazy-hydrated as small islands. The design system leans on calm typography and a single green accent — the same way this site does.
